
# 🎵 Spotify Clone

A responsive Spotify-inspired web music player built using HTML, CSS and Bootstrap

## 🚀 Features
- Play/Pause music
- Song progress bar
- Interactive UI
- Responsive design

## 🛠️ Tech Stack
- HTML5
- CSS3
- Bootstrap

## 📂 How to Run
1. Clone the repository
2. Open index.html in your browser
